自己在想什么呢

12月 26, 2017 | 这一天被 788 字所记录
内心

不知道这一页是当做日记写还是当做内心独白一样的东西来写呢…
说到底这里只是用来让自己学会说出一些东西的地方的呀，叫日记本，也只是希望自己能够每天都写一页呢。

自己在想什么呢…猫猫自己也不知道呢，你们都在努力，你们都在写年终总结，你们都在做着点什么…
感觉好羡慕，好嫉妒。猫猫不会写东西呢，不要说是什么年终总结，就算是猫猫自己的主站和过去的博客，写的东西也很少很少不是吗。何况，写的日记大概读起来也是很难受的语法什么的吧…
如果有人来仔细阅读的话，希望不要讨厌呢。

每次看到群里的你们，有大学，有喜欢的东西可以学，会那么多东西，学的那么好，那么厉害，会社交，有那么多朋友，还能那么开心…
不知道自己应该怎么办呢。
什么都没有的自己，也不知道怎么做…那些希望的东西大概早就不存在了不是吗？如果在的话，那也是在另一个平行世界了吧。
时不时就会负能量爆炸的自己…时不时就会哭出来的自己…自己到底在想什么呢…
猫猫也希望能找回以前的那些东西的呀…可是…失去了好多呢不是吗。
猫猫是一只坏猫呢，是一只会伤害别人和忽视别人关心的坏猫呢…

大概是因为自己只想要安全感吧。
和你在一起，感觉自己变脆弱了好多呢，因为你就是一切呀，你在的时候给猫猫的那些温暖，就足够了呢，不会去乱想，不会去做什么，因为感觉很安全呢。

比起之前自己已经好一些了呢，只是会嫉妒你们…嫉妒你们有的这些东西，这些自己想要拥有却抓不到的东西。
明明自己曾经也有的呀，为什么家里人要这样对待猫猫呢…

现在的话，此时此刻的话，想着的东西就是“纠结于自己应该做什么才能找回以前的那些东西…”
今天有打开 Logic 找到自己喜欢的音色，是八音盒。


不过记忆中只用过两次 Logic，而且当时猫猫的 FL Studio 自动化操作和自己的编曲能力还非常糟糕，也没有做出什么东西来呢。
蛮惊艳于 Logic 的预设音效库呢，超级棒的那种，虽然还是比不上那些上百 GB 的精品交响乐音效，但是对于小作品什么的已经很足够了呢。
不知道能做什么呀…试试看做一个八音盒音色的东西呢。
毕竟，喜欢人偶和八音盒在一起的东西…不知道为什么，异常喜欢。大概是喜欢童话吧？

剩下的就是…难受了吧。
真的好羡慕你们呢。
